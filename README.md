# tbx-windows-ui-executor

Windows 11 **tray** Executor: exposes **screenshot + window + input** primitives over **tailnet** (HTTP + Bearer token).

- Controller: `mf-kvm01 (100.64.0.1)`
- Executor: runs on `yc-tbx` (Win11)
